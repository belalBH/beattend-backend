# 🔥 دليل تحويل جسر برو كامل إلى PHP Laravel

أهلاً بك! تم تحويل النظام بالكامل وجاهز للتصدير والتثبيت على خادمك المحلي أو السحابي باستخدام **PHP Laravel** مع واجهة **React** التفاعلية باستعمال **Laravel Vite**.

---

## 📂 الهيكل المالي والمجلّدات الجديدة لـ Laravel
تم إنشاء وتوليد الملفات اللازمة لنظامك في مجلد `/laravel`:
- **قواعد البيانات (Migrations):** داخل `database/migrations` (الشركات، الأقسام، الموظفين، نطاقات الفروع الجغرافية).
- **النماذج الذكية (Model Relationships):** في `app/Models` (الشركات والموظفين مع العلاقات `hasMany` و `belongsTo`).
- **المتحكمات (Controllers):** في `app/Http/Controllers` لتقديم استجابات JSON سريعة متوافقة مع واجهات React.
- **مسارات الويب والـ API:** في `routes/api.php` و `routes/web.php` (تتضمن فحص الإحداثيات الجغرافية عبر معادلة هافيرسين الجيبية تلقائياً).
- **القوالب (Blade Views):** في `resources/views/app.blade.php`.

---

## 🛠️ خطوات التشغيل والتثبيت الفوري

### 1️⃣ المتطلبات الأساسية
تأكد من تنصيب التالي على جهازك:
- **PHP** (الإصدار 8.1 أو أحدث)
- **Composer** لإدارة حزم PHP
- **Node.js & NPM** لإدارة حزم الواجهة الأمامية

### 2️⃣ تثبيت حزم واجهات الخلفية والأمامية
قم بفتح المجلد واكتب الأوامر التالية:
```bash
# تثبيت حزم Laravel الخلفية
composer install

# تثبيت حزم React الأمامية
npm install
```

### 3️⃣ إعداد البيئة وقواعد البيانات `.env`
قم بنسخ ملف الإعداد الافتراضي وضبط إعدادات قاعدة البيانات الخاصة بك (MySQL / PostgreSQL):
```bash
cp .env.example .env
```
افتح ملف `.env` وقم بتعديل بيانات الاتصال:
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=jesr_db
DB_USERNAME=root
DB_PASSWORD=your_password
```
ثم قم بتوليد مفتاح حماية التطبيق:
```bash
php artisan key:generate
```

### 4️⃣ ترحيل جداول قواعد البيانات (Migrations)
قم بترحيل الجداول لتأسيس هيكل الشركات والموظفين والتحكم في النطاقات الجغرافية:
```bash
php artisan migrate
```

## 📍 كيف يعمل فحص النطاق الجغرافي ذكياً في Laravel؟
لقد قمنا بتطوير ميزة تسجيل الحضور الحقيقية في `routes/api.php` باستخدام **معادلة هافيرسين الجيبية (Haversine Formula)** لحساب المسافة الدقيقة بين الموظف والفرع بالأمتار:

```php
// يتم إرسال إحداثيات الموظف (Latitude, Longitude) عبر التطبيق
// ويقوم الخادم بالتحقق من مسافته بالنسبة لفروع شركته المخزنة
$distance = $angle * $earthRadius; // النتيجة بالأمتار

if ($distance <= $loc->radius) {
    // الموظف متواجد داخل الدائرة المسموحة!
}
```

## 🚀 تشغيل خادم التطوير والواجهات معاً

لتشغيل خادم Laravel الخلفي وخادم Vite لبناء واجهة React التفاعلية، افتح نافذتين في وحدة التحكم (Terminal):

**النافذة الأولى (خادم Laravel API):**
```bash
php artisan serve
```

**النافذة الثانية (خادم Vite لبناء React):**
```bash
npm run dev
```

الآن افتح المتصفح على الرابط الافتراضي: `http://127.0.0.1:8000` لمشاهدة جسر برو يعمل بالكامل على خادم Laravel الخاص بك!
